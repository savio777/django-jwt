# django-jwt

### steps init project:
- `python3 -m venv env`
- `pip install -r ./requirements.txt`
- `django-admin startproject backend`
- `cd backend && python manage.py startapp api`
- `python3 manage.py makemigrations`
- `python3 manage.py migrate`
- `python3 manage.py runserver`
